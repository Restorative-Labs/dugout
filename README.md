# ⚾ Dugout

**Film. Freeze. Fix.** — A film room for parent-coaches.

Dugout helps a parent run a real hitting session with their kid, even without a coaching background: film a swing, slow it down frame by frame, chalk coaching lines right on the footage, and follow checkpoint cards that show a real shadow-silhouette MLB swing with the key positions marked.

**Live demo:** https://restorative-labs.github.io/dugout/

## What's inside

- **Film** — record or load a swing, ¼/½ speed playback, frame-by-frame stepping
- **Chalk kit** — draw lines, arrows, circles, and measured angles on any frozen frame
- **Checkpoints** — six cards (Stance → Load → Stride → Hips Fire → Contact → Finish), each with an animated shadow swing that slows down and highlights its phase, plus coach-voice and kid-voice cues
- **Compare** — two swings side by side with frame-nudge alignment and synced playback
- **AI Coach** — frame-window analysis returning one high-leverage fix, a kid-friendly cue, a drill, and on-film annotations *(available in the Claude-hosted version; this static demo runs everything else)*
- **Scouting notes** — persist locally between sessions

## Running it

It's one self-contained HTML file — no build, no dependencies, works offline.
Open `index.html` in any browser, or serve via GitHub Pages.

## Roadmap

Dugout Hitting is the first module. The vision: a platform AAU teams use for
position-by-position skills training — pitching, catching, and infield modules
with per-player film history and progress tracking.
